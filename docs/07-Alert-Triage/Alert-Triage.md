# Phase 7 — Alert Triage & Investigation

## Objective

Perform Security Operations Center (SOC) alert triage by investigating security events within the Wazuh SIEM, analyzing alert metadata, validating endpoint telemetry, and documenting the incident investigation workflow.

---

## Activities Performed

- Monitored security events using the Wazuh Threat Hunting dashboard.
- Filtered and reviewed endpoint security alerts generated by the monitored Windows system.
- Investigated alert metadata, including event details, rule information, severity levels, and compliance mappings.
- Examined raw JSON event logs to validate collected telemetry and security event attributes.
- Reviewed surrounding events to understand the context and sequence of security events during the investigation.

---

## Tools Used

- Wazuh SIEM
- Wazuh Threat Hunting
- Sysmon
- Windows 10 Endpoint
